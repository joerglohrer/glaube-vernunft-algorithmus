Mitmachen — gemeinsam besser werden

Diese Materialien leben davon, dass andere sie nutzen, anpassen und zurückspielen. Egal ob Tippfehler, didaktische Idee, neuer Entwurf oder ganz andere Schulart — jede Form von Beitrag ist willkommen. Beiträge stehen ebenfalls unter CC0.

  • Kleine Korrektur? Auf GitHub im Browser bearbeiten („Edit"-Stift oben rechts in der Datei) — daraus wird automatisch ein Pull Request.
  • Größere Erweiterung? Repository forken, Branch anlegen, Pull Request öffnen.
  • Idee oder Frage? Issue auf GitHub eröffnen.
